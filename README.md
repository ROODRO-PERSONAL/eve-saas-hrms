# Mini HRMS Module (Laravel API)

## Overview
This project is a mini HRMS (Human Resource Management System) built in Laravel 10, designed to demonstrate SaaS-oriented architecture with multi-tenant support.

---

## Features
- Employee Management (CRUD)
- Attendance Management (Clock-in/out, total hours calculation)
- Leave Management (Apply, approve/reject)
- Multi-tenant support using `company_id`
- API authentication using Laravel Sanctum

---

## Tech Stack
- PHP 8.x, Laravel 10
- MySQL
- Laravel Sanctum (API authentication)
